# Wanderlust

## Overview
Wanderlust is a full-stack hotel booking and listing platform inspired by Airbnb. Users can explore properties, create and manage their own listings, upload images, and leave reviews. The application includes authentication, authorization, and a responsive interface for a smooth user experience.

## Features
- User authentication and authorization
- Add, edit, and delete hotel/property listings
- Upload property images
- View detailed listing information
- Leave and manage reviews
- Responsive and user-friendly interface
- Secure backend with protected routes

## Installation
1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create a `.env` file and add the required environment variables.
4. Start the server:
   ```bash
   npm start
   ```
5. Open `http://localhost:3000` in your browser.

## Usage
- Sign up or log in to your account.
- Browse available hotel listings.
- Add your own property listings.
- Edit or delete your listings.
- Leave reviews on listed properties.
